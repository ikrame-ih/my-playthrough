Modelo lógico — tablas principales

Vista HTML del esquema alineada con docs/schema.sql. La tabla usuarios incluye avatar_id (robots predefinidos robot-0 … robot-9).

usuarios
idSERIAL PK
nombre_usuarioVARCHAR(255) NOT NULL
emailVARCHAR(255) UNIQUE NOT NULL
password_hashVARCHAR(255) NOT NULL
rolVARCHAR(50) DEFAULT 'user'
avatar_idVARCHAR(32) NOT NULL DEFAULT 'robot-0'
fecha_registroTIMESTAMP DEFAULT NOW()
catalogo_juegos
idSERIAL PK
tituloVARCHAR(512) NOT NULL
url_imagenTEXT
rawg_idINTEGER UNIQUE
steam_app_idBIGINT UNIQUE
created_atTIMESTAMPTZ
juegos
idSERIAL PK
usuario_idFK → usuarios(id) CASCADE
tituloVARCHAR(255) NOT NULL
estadoVARCHAR(50) NOT NULL
plataformaVARCHAR(100)
puntuacionINTEGER
horas_jugadasINTEGER DEFAULT 0
comentarioVARCHAR(1000)
url_imagenTEXT
catalogo_idFK → catalogo_juegos(id) SET NULL
juego_comentarios
idSERIAL PK
juego_idFK → juegos(id) CASCADE
usuario_idFK → usuarios(id) CASCADE
parent_idFK → juego_comentarios(id) CASCADE
cuerpoTEXT NOT NULL
fecha_creacionTIMESTAMPTZ DEFAULT NOW()
